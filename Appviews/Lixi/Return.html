QUY TRÌNH NHẬN HOÀN/ ĐỔI TRẢ

Bước 1: Nhận hàng hóa trả về từ Shipper:

  • Đồng kiểm số lượng, tình trạng của đơn hàng.
  • Lập biên bản nếu hàng hóa hư hỏng, có vấn đề để làm căn cứ xác định bồi thường.

Bước 2: Tìm kiếm order/ Tracking Number trên hệ thống → Customer Returns → Create New Customer Return

Bước 3: Xử lý các thông tin được yêu cầu:

  • Chọn các sản phẩm có trong đơn hàng trả về
  • Warehouse:
    • Packing Warehouse: (SG Warehouse, HN Warehouse) sản phẩm có thể lưu kho bán tiếp.
    • Damaged Warehouse: Hàng hư hỏng.
    • Warranty Warehouse: Hàng bảo hành, đổi trả.
  • Chọn lý do hoàn trả hàng.
  • Xác nhận Tạo Customer return để tạo mã Return cho đơn hàng hoàn trả.

Bước 4: Vào Mã Return tương ứng với đơn hàng trả đã được tạo trên hệ thống → chọn Reimburse Type → Cập nhật Reimbursement → Reimburse.
