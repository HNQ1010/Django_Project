II) LIST:

1) Adding elements into a list

    .append() : insert (all -if there is a list, tuple)elements into the last of the list

    .extend() : insert elements (one-by-one if there is a list or tuple)into the last of the list

    .insert() : You can add a new element at any position in your list by using the insert()

    del :If you know the position of the item you want to remove from a list, you can use the del statement.

    .pop():

    The pop() method removes the last item in a list, but it lets you work with that item after
    removing it. The term pop comes from thinking of a list as a stack of items and popping one item
    off the top of the stack. In this analogy, the top of a stack corresponds to the end of a list.

    Popping Items from any Position in a List :You can actually use pop() to remove an item in a list at any position by including the index of the item you want to remove in parentheses.

    .remove() :If you only know the value of the item you want to remove, you can use the remove() method.

    If there’s a possibility the value appears more than once in the list, you’ll need to use a loop to determine if all occurrences of the value have been removed.
